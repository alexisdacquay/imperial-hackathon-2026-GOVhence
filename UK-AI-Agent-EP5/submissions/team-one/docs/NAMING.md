## 0. The headline correction: copyright ≠ what you want

You asked about **copyright** for the names. Copyright is the wrong legal tool:

- **Copyright** protects *creative works* (code, text, art, music). Both the US Copyright Office and
  the UK IPO **refuse to register names, titles, slogans, or short phrases.** You cannot copyright
  "GOVhence" or "MEM-0".
- Your **source code is already protected by copyright automatically** the moment it's written — no
  registration needed (registration mainly helps with US litigation damages).
- The right tool for a **brand name** is a **trademark** (™ unregistered / ® registered). The rest
  of this document is about trademarks.

## 2. Name-strategy memo — the recommended move

**Register the family mark led by GOVhence; do not try to register "MEM-0" head-on against Mem0.**

Concretely, three options, best first:

1. **(Recommended) File `GOVHENCE` as a word mark.** Distinctive, clear, registrable. This protects
   the brand that actually differentiates you. Keep "MEM-Ø/MEM-0" as an internal product/edition name
   used *under* the GOVhence brand, not as the registered mark.
2. **File `GOVHENCE MEM-Ø` as a combined mark.** Registrable because GOVHENCE carries the
   distinctiveness, but the MEM-Ø element is weak and an examiner may require a *disclaimer* of it
   (i.e. you get no exclusive rights to "MEM" alone). Fine, but option 1 is cleaner.
3. **Restyle the sub-name to remove the clash** (e.g. `GOVhence Mem-Ø` → something less identical to
   "Mem0", or drop the product suffix entirely). Worth considering if MEM-0 matters to you long-term.

**Avoid:** filing "MEM-0" or "Mem0" alone in software/AI classes. High refusal + opposition risk,
wasted fees.

**Word mark vs logo (figurative) mark:** register the **word mark** ("GOVHENCE" as plain text)
first — it protects the name in any font/styling and is the most valuable. The slashed-Ø stylisation
is a *logo* and would be a separate **figurative** filing; only do that if the visual identity itself
becomes worth protecting.

## 3. How-to-file guide

### Which office?
- This is a **UK** hackathon, so **UK IPO** is the natural first filing. (One filing = UK only.)
- Software brands usually also want the **US (USPTO)** and **EU (EUIPO)** if they'll operate there.
- Going multi-country later is easiest via the **Madrid Protocol** (WIPO) — one international
  application that designates multiple countries, usually filed *after* you have a "home" (e.g. UK)
  application as the base.

### Which classes? (the "Nice" classification)
A trademark is registered for specific **classes** of goods/services. For governed-memory software
you'd typically file in:
- **Class 9** — downloadable software / computer programs.
- **Class 42** — SaaS, PaaS, software development, hosting, technology services. (Most likely your
  core, since this is delivered as a service.)
Filing both 9 and 42 is common for a software product.

### UK IPO — cost & timeline (2025/26)
- **£200** for the first class, **+£50** per additional class → **~£250** for classes 9 + 42.
- Optional **"Right Start"**: pay ~half up front for an examiner's opinion, then the rest if you
  proceed — a cheap way to test viability.
- Online application ~30 min. Examination in ~2–3 weeks; if unopposed, registration ~4 months
  (includes a 2-month publication/opposition window).
- File at: gov.uk → "Register a trade mark".

### USPTO — cost & timeline (2025 system)
- As of **Jan 18 2025**, TEAS was replaced by **"Trademark Center"**; one **Base application
  ≈ $350 per class**, with surcharges if you use free-text (non-standard) goods/services descriptions
  or overly long descriptions. Using the standard ID Manual wording avoids surcharges.
- First examiner action in ~3–4 months; total ~8–12 months if smooth, 12–18 if an office action
  issues. The USPTO's AI-driven confusion search is now *more* sensitive — another reason the MEM-0
  clash is risky there.
- US filings generally need a **US-licensed attorney** if the applicant is foreign-domiciled.

### DIY vs attorney
- **DIY** is feasible for a single, clearly-distinctive UK word mark (GOVHENCE) using standard class
  wording — lowest cost.
- **Use an attorney** when: filing in the US (often mandatory for non-US applicants), filing
  internationally, anything touching the Mem0 conflict, or if a logo/figurative mark is involved.
  A proper clearance search + filing is where their value is highest.
